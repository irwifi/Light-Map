Features of the Map

EASILY INTEGRABLE

Can be integrated to any platform, framework or system with only a single line of code.

EASILY CUSTOMIZABLE

Easily customizable to be completely tailored to your specific project.

RESPONSIVE

Adjusts to any screen size beautifully.

MOBILE READY

Works well in mobile and tablets.

HIGHLY INTERACTIVE

User can interact with the map in many ways and such interactions can be used to trigger events to perform various actions like display information or summary, enlarging the map etc.

SPECIAL FEATURE

Usually the easy customization and easy integration do not go together. If it can be easily integrable then it is not easily customizable and vice versa. But Light Map has both features and that is what makes it special than other similar maps.
